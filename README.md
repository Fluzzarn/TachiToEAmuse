# TachiToEAmuse
Converts a Tachi PB list to eAmuseCSV for SDVX

Invocation:

Navigate to Tachi -> SDVX -> Scores -> All PB's
Press "Export Table (JSON)" at the bottom of the table (You might need to turn on developer mode?)
Paste into a local file
Run this passing in the file

CSV saved as ~/scores.csv
